# Preview Orderia Project

This workflow describes the steps to start the development server and preview the application.

## 1. Start Dev Server
Run the development server with Turbopack enabled.
```bash
npm run dev
```

## 2. Access Preview
By default, the application will be available at:
`http://localhost:3000`
