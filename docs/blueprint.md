# **App Name**: Orderia

## Core Features:

- Customer Onboarding & Secure Login: Facilitates quick and secure customer login via PIN or password, supporting multi-supplier connections and status approval by wholesalers. Features 'remember me' for ease of access.
- Intelligent Order Creation: Customers can easily browse products, adjust quantities with intuitive +/- buttons, and leverage an intelligent algorithm for suggested order quantities based on ideal stock, current inventory, and previous orders.
- Customer Warehouse Inventory: Enables customers to update their current stock manually or by scanning product barcodes. The system then automatically calculates and suggests optimal order quantities.
- Order History & One-Click Reorder: Provides customers with a comprehensive history of past orders and allows for rapid reordering with a single click, fulfilling the 'less than 2 clicks' goal.
- Wholesaler Admin Dashboard: A desktop-first interface for wholesalers to manage orders, customer accounts, assign products, set ideal stock levels, and oversee pending customer connection requests.
- Wholesaler Internal Inventory: Allows the wholesaler to manage their own warehouse stock, including updating quantities via barcode scanning, tracking inflows/outflows, and receiving low stock alerts.
- Excel Export for Reporting: Enables wholesalers to export order and inventory data into Excel format, customizable by period (order, day, week) or customer, for detailed analysis.

## Style Guidelines:

- Color scheme: A premium dark mode palette to create a sleek, sophisticated atmosphere.
- Primary color: A robust, active blue (#6999EB) to signify core interactive elements and status indicators.
- Background color: A charcoal black base (#18181A) providing depth and strong contrast, with a subtle cool undertone.
- Accent color: Vibrant 'neon blue' (#00AFEE) for critical actions, highlights, and emphasis, ensuring strong visual appeal and attention-grabbing elements.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, techy, and bold statement.
- Body text: 'Inter' (sans-serif) for exceptional readability and clear typography, optimized for screen display and mobile-first content.
- Mobile-first spacing and 'rounded 2xl cards' are implemented for optimal user experience on small screens, offering clear content segmentation and intuitive navigation.
- Large, finger-friendly 'tap buttons' to minimize mis-taps and simplify user interaction, contributing to an 'ultra simple UX' and 'less than 2 clicks' functionality.
